# Theft Alarm System using Arduino + Ultrasonic Sensor

This project detects movement using an ultrasonic sensor connected to an Arduino.  
When an intruder is detected within a certain distance, the Arduino:

- Blinks alert LEDs  
- Sends "ALERT" to the computer  
- Python script sends an EMAIL notification
